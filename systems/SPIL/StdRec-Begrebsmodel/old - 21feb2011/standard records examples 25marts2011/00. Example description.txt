INFORMATION ON EXAMPLES FILES - 15. February 2011

In this folder you find examples of the XML files required by the Danish Gambling Authority. 

The examples have been generated automatically and illustrate the syntax and data types.

What the actual content should be of each data element is described in "Begrebsmodel for spil kontrol - english - 06okt2010.pdf", which you can find on the Gambling Authority's website. 

We hope that these examples can be helpful.
