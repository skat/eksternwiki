Html5 input typer

Number input typen

Denne type kan bruges i moderne browsere for at vise et input felt med pile i højre side, hvor brugeren kan kan trykke op eller ned. Ligeledes kan der ved fokus på feltet bruges tastaturets piletaster for at øge eller sænke tallet i input feltet.

Hvis en browser ikke understøtter ovenstående tag, skal der bruges javascript for at emulere denne input type, eksempelvis ved brug af en klasse som kan referes.
